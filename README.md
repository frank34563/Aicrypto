# AiCrypto Investment Bot

Telegram bot with 1.5% daily profit, admin approvals, SQLite.

## Features
- 2-column main menu
- Deposit & Withdraw with admin approval
- 1.5% daily profit (auto)
- Full history
- Wallet settings
- Support button
